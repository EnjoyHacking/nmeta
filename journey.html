The Journey

The idea for nmeta came from experience of a problem in enterprise networks, where QoS classification was becoming unwieldy, mainly due to an increasing number of flow types.

Post-graduate study at Victoria University created an opportunity to investigate potential solutions. A COMP489 project was the vehicle, under supervision of Prof Winston Seah and Dr Bryan Ng.

The code is now with the open source community, to see where it can be taken.
